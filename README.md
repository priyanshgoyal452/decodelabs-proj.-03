# Retail Sales Data Analysis using SQL

## Overview

This project demonstrates SQL-based data analysis on a retail sales dataset using Microsoft SQL Server (SSMS). The objective was to analyze sales data by writing SQL queries and extracting meaningful business insights using filtering, sorting, grouping, and aggregate functions.

## Tools Used

* Microsoft SQL Server Management Studio (SSMS)
* SQL

## Dataset

The dataset contains retail sales transactions with information such as:

* Order ID
* Date
* Customer ID
* Product
* Quantity
* Unit Price
* Payment Method
* Order Status
* Coupon Code
* Total Price

## SQL Concepts Covered

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* COUNT()
* SUM()
* AVG()
* MAX()
* MIN()
* TOP

## Analysis Performed

* Total number of orders
* Total revenue
* Average order value
* Highest and lowest order value
* Top selling products
* Revenue generated by each product
* Payment method analysis
* Order status analysis
* Coupon usage analysis
* Highest value orders

## Conclusion

This project demonstrates the use of SQL for cleaning, querying, and analyzing retail sales data. The analysis provides insights into sales performance, customer purchasing behavior, payment preferences, and product performance using fundamental SQL operations.
